| 試験項目（規格） | API SM（2004） | API SN（2010） | API SP（2020） | API SQ（2024） |
|----------------------|----------------|----------------|----------------|----------------|

### 規格の進化

- **API SM → SN**：VH、IVB試験を導入し、摩耗と堆積物制御を向上させた。
- **SN → SP**：LSPIおよびタービン堆積物試験を追加し、GDIターボエンジンを完全にサポートした。
- **SP → SQ**：すべての試験基準を強化し、E85、GPF/DPF、ハイブリッド車をサポートし、ILSAC GF-7に対応した。
