## 選定背景

数年前から、自宅の輸入車に順次 [GreenRun リン酸鉄リチウムバッテリー](/ja/2025/greenrun.html) を搭載しており、使用感、寿命ともに非常に満足しています。最近、友人の HS 1.5T の純正バッテリーが劣化し始め、内部抵抗が増加したため、調査の結果、リン酸鉄バッテリーへの交換を決定しました。

HSの純正バッテリーは風帆 6-QW-63MG であり、**LN2**規格に属します。HSのバッテリーベースはより大きなLN3規格に対応していますが、LN3規格に変更すると、純正の断熱カバーが使用できなくなるため、LN2規格を維持することにしました。

台湾において、LN2規格と日規B24、D23は同様の問題を抱えています。市場に出回るリン酸鉄バッテリーの選択肢が限られており、**ほとんどの製品が個人業者からのものであったり、製造元の情報が明記されていなかったりします**。中国市場には多くの12Vリン酸鉄ブランドがありますが、一部のブランドは容量の虚偽表示や粗悪な作りだと批判されることがあります。比較検討の結果、最終的に、長年リン酸鉄スターターバッテリーに携わり、欧米ブランド向けにリチウムバッテリーのOEM/ODMソリューションを提供しているメーカーである **弘毅 (Houny)** を選択しました。

[![弘毅 Houny 金階 LN2 85Ah 車用スターターリン酸鉄バッテリー規格表](/images/LN2_HY_Type.webp)](/images/LN2_HY_Type.webp)

弘毅が2025年に改版して以来、製品ラインは金階、藍階、緑階の3つのシリーズに分かれており、今回購入したのは85Ahの金階モデルです。同シリーズのLN2規格では100Ahも選択可能ですが、友人のHSは純粋な純正設定で、休日の使用のみであり、重度の電装系改造の必要がないこと、さらにリン酸鉄バッテリーの放電深度 (Depth of Discharge, DoD) が深いため、実用容量が鉛蓄電池の1.6倍となるため、**85Ahのリチウム鉄は実質的に130Ahの鉛蓄電池に相当し、十分すぎるほどの容量です**。

## 取り付け手順

### 準備工具

| 工具名 | 用途と備考 |
| :--- | :--- |
| **10mm [L字ソケット](/images/L-socket.webp)** | バッテリーベースの固定ネジ、およびプラス・マイナス端子を外すのに必須 |
| **[OBDメモリーセーバー](/ja/2025/memory-saving-battery.html)** | 電源を切らずに交換するために使用し、車載コンピューターやオーディオの設定が失われるのを防ぐ。 |
| **補助電源** | 任意の12Vバッテリーまたはジャンプスターター用電源を、OBDメモリーセーバーへの給電に使用する。 |
| **[バッテリーハンドル](/images/battery_handle.webp)** | 取り出しの補助（純正バッテリーにはハンドルがないため、体力節約のため準備を強く推奨します）。 |

### 取り付けステップ
> 以下は実際の作業手順の記録であり、取り付け完了後にエラーコードはありませんでした。

[![MG HS 1.5T カーバッテリー DIY 電源切断取り付け手順図解](/images/LN2-instruction.webp)](/images/LN2-instruction.webp)

| ステップ | 作業説明 | 注意事項と詳細 |
| :---: | :--- | :--- |
| **1** | **メモリーセーバーの接続** | 補助電源を**OBDメモリーセーバー**に接続し、車内のOBDポートに差し込む。 |
| **2** | **車両のスリープ化** | すべてのドアを閉め、車両コンピューターが完全にスリープ状態になるまで5分間待機する。 |
| **3** | **ベースの取り外し** | 10mmソケットを使用してバッテリーベースの固定ネジを緩め、ネジとプレーンプレートを取り外す。 |
| **4** | **作業スペースの確保** | 古いバッテリーを**車頭方向へ押し出す**ことで、ラッチから離し、プラス極側の操作スペースを広げる。 |
| **5** | **端子の切断** | バッテリーの**マイナス極**から順に緩め、次に**プラス極**の端子を外す。 |
| **6** | **バッテリー交換** | [バッテリーハンドル](/images/battery_handle.webp)を使用して古いバッテリーを取り出し、新しいリン酸鉄リチウム電池をセットする。 |
| **7** | **端子の再接続** | 順に**プラス極**から、次に**マイナス極**の端子を締め付ける。 |
| **8** | **新バッテリーの固定** | 新しいバッテリーを**運転席側方向へ押し込み**、ベースの固定ラッチがしっかりと嵌まっていることを確認する。 |
| **9** | **ベースの再装着** | ベースプレートとネジをしっかりと元に戻して締め付ける。 |
| **10** | **取り付け完了** | OBDメモリーセーバーを取り外し、エンジンを始動させてすべて正常であることを確認する。 |

## 完了報告と所感

今回の実際の取り付け時間は約15分で、MG HS 1.5Tのエンジンルームはスペースが十分あり、交換しやすい車種でした。

[![MG HS 1.5T エンジンルームに弘毅 LN2 リン酸鉄リチウム電池を実際に視認](/images/LN2_HY_Install1.webp)](/images/LN2_HY_Install1.webp)

車友がリン酸鉄リチウム電池への交換を決めた主な理由は、以下のコアな優位性によるものです。

| 特長 | 詳細説明 | 期待される効果 |
| :--- | :--- | :--- |
| **充放電性能** | [充電効率が高い](/images/mobile01-d8c9d0d9da6aecd839d3e3fbe4be1398.webp)ため、エンジンの負荷を効果的に軽減し、[放電反応が速い](/ja/2025/lithium-vs-lead-analysis.html)ため、電気系統の動作がより安定する。 | 車載発電機の寿命延長、およびアイドリング品質の改善。 |
| **環境適応性** | [高温低温での安定性](/images/mobile01-8aeff577382fa233baa6f5da8eaead0b.webp)を備え、リン酸鉄リチウム構造が安全である。 | 高温に耐え、爆燃のリスクがないため、エンジンルームの電力使用における安全性が向上する。 |
| **物理的特性** | 重量約50%減（純正約17kgに対し、リン酸鉄はわずか8kg）、自己放電率が極めて低い。 | 軽量化によるハンドリング性能の向上、**極めて低い自己放電率は休日利用に非常に適している**。 |
| **燃費検証** | [高い充電効率によりエネルギー損失を減少させる](/ja/2025/car-bms.html)ため、発電機が充電を行う時間が大幅に短縮される。 | [研究](/images/mobile01-be90eec739eb64e5d45690809d60a24e.webp)によると、交換後には車両の燃費性能を実質的に最適化できる。 |
