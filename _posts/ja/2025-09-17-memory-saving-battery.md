# バッテリー交換時に給電を維持する（記憶保持）

> ISO 26262やUN R13/79などの国際規格に基づき、自動車の電子システムは【通電停止または再通電】した際に、「安全な状態」に戻る必要があります。

トヨタやホンダの場合、通電停止しても通常は時計のリセットだけで済みます。しかし、車両を一定期間使用している場合、**いくつかの記憶設定が失われる可能性**に注意が必要です。

| 記憶パラメータ | 記憶クリア後の現象 |
| :--- | :--- |
| <nobr><B>スロットル開度</B></nobr> | ECUは対応するカーボン堆積状態の補償記憶を失い、工場出荷時の初期開度に戻ります。これにより実際の吸気量が不足し、**アイドリングが不安定になったり、激しく振動したり、停車時やクリープ走行中に瞬間的にエンジン停止する**可能性があります。 |
| <nobr><B>アイドル制御</B></nobr> | アイドル空気流量の学習値を失います。システムが再学習を完了するまでの間、**アイドリングが不安定になったり、回転数が異常に変動したり**する状況が発生します。 |
| <nobr><B>トランスミッション変速ロジック</B></nobr> | 過去の運転習慣やアクセル開度記録に基づいたパラメータをクリアします。短期間では、トランスミッションが**変速タイミングを変えたり、ショック感が顕著になったり、繋ぎ目が硬く感じられたりする**ため、コンピューターに再適応させるために一定時間走行する必要があります。 |
| <nobr><B>燃料補正値</B></nobr> | インジェクション量の長期修正値がゼロに戻ります。ECUが酸素センサーのデータに基づいて最適な空燃比モデルを再構築するまでの間、短期間に**燃費が悪化したり、エンジン出力が不安定になったり**する現象が起こる可能性があります。 |
| <nobr><B>ステアリング角度センサー</B></nobr> | ステアリングの中心点校正基準を失います。ダッシュボードの故障ランプが点灯したり、**ESP（車体安定制御）やABS（アンチロック・ブレーキシステム）などのシステムが正常に作動しなくなる**可能性があります。 |

> これらの設定は一定時間走行することで自動的に再学習されますが、それでも車両異常を引き起こす可能性があります。

通電停止によって上記のような不便が生じるにもかかわらず、なぜ多くの純正整備マニュアルやSOP（標準作業手順）では「通電停止での交換」となっているのでしょうか？
主な理由は、**メーカー純正の専用診断コンピューターが用意されているから**です。通電停止後、ディーラーの技術者は故障コードを即座にクリアしたり、センサーの校正を行ったり、さらにはシステムが盗難ロックした場合でも、防犯キーの同期と認証を再実行することができます。

これに対し、一般の店舗やDIYユーザーは**メーカー純正権限のコンピューターを持たないため、「記憶保持器」を使用して通電停止せずに交換を行うのが最も安全な方法**です。これにより、ECUの学習パラメータを維持できるだけでなく、IMMO（イモビライザー）のエラーや電子モジュールの損傷を防ぎ、ディーラーに持ち込んだ際の「開通」や「再書き込み」のリスクを低減できます。

## 通電停止なしでのDIY注意事項：

## 1. プラス端子は依然として給電されている（最重要）

通常の場合、バッテリーを外すとプラスのクランプは無電の金属になります。
しかし、OBD-IIインターフェースやシガーソケットから外部電源（記憶保護器）を介して接続した場合、プラスのクランプは12V電圧を維持したままとなります。

- **リスク**：緩んだプラスのクランプが車体、エンジン、またはショックタワーなどの金属部分に接触すると、ショートを引き起こし、記憶保護器やOBD-IIインターフェースのヒューズが焼けるだけでなく、ECU（コンピューター）自体を損傷させる可能性があります。
- **解決方法**：バッテリーを移動する前に、すぐに**厚手の乾いた布、ゴム手袋、またはビニール袋でプラスのクランプを覆い、テープやストラップで固定する**などして、金属への接触を防いでください。
---

## 2. ドアを開けたり、車内の電気機器を操作しないこと

- **リスク**: ほとんどのメモリプロテクターが供給する電流は非常に低いです。ドアを開けると、ドームライトやダッシュボード画面が点灯し、瞬間的な大電流が必要となり、メモリプロテクターが空電したり、ヒューズが飛んだりして、車両全体の通電停止を引き起こす可能性があります。
- **解決方法**: 事前に運転席の窓を下げ（ロックしないように）、ヘッドライト、ラジオ、エアコンを消し、ドアは常に閉めた状態を維持してください。

---

## 3. 12Vソケットの確認（シガーソケット方式を使用する場合）

- **リスク**: 多くの現代の車両では、エンジン停止後、点火装置/12Vソケットへの電源が遮断されます。もしキーを取り外した後にソケットに電気が流れていなければ、メモリプロテクターは逆方向への給電を行うことができません。
- **解決方法**: まず、鍵を差し込まない状態でソケットに電気が流れているかテストしてください（例：携帯電話の充電器を挿入）。もしソケットが通電しない場合は、OBDIIインターフェース方式に変更する必要があります。

---

## 4. 接続の安定性

- **リスク**: 重いバッテリーを移動させる際、メモリプロテクターのクランプやプラグは接触不良を起こしやすいです。一瞬でも接続が切れると、記憶が失われます。
- **解決方法**: メモリプロテクターのプラグがしっかりと固定されていることを確認し、配線は整理して、作業中に引っかからないように注意してください。

---

## 5. 取り付け後：BMSのリセット

通電を維持することで各設定を保持できても、バッテリー管理システム（BMS）の処理が必要です。

- **原因**: 通電停止は記憶を保存するだけであり、「新しいバッテリーに交換された」という情報を車両に伝えることはできません。システムは古いバッテリーがまだ使用されていると認識します。
- **解決方法**: 新しいバッテリーを取り付けた後、診断ツールまたは特定のボタン操作手順を通じて、新しいバッテリーを「登録」または「コーディング」し、発電機が正しく充電できるようにする必要があります。
