faq:
  - question: "API認証の中で、どれがLSPI問題に対応していますか？"
    answer: "API SN Plus と API SP はどちらも LSPI テストの合格を要求しています。最新の API SQ (ILSAC GF-7) は LSPI 防護、タイミングチェーン耐摩耗性、燃費性能をさらに強化し、GPF 保護のため硫酸灰分を制限しています。"
  - question: "メーカー認証と API 認証の違いは何ですか？"
    answer: "メーカー認証（例：MB 229.5）は通常、HTHS、TBN、Noack などのパラメータに対してより具体的な基準を設けています。例えば MB 229.5 は耐摩耗性基準が高いですが LSPI テストはありません。API SP/SQ は耐摩耗性基準が低めですが LSPI テストがあります。理想的には、オイルがメーカー認証と API SP/SQ 認証の両方を持つことで、最適な耐摩耗性と LSPI 防護を提供できます。"
  - question: "エンジンオイルが本当にメーカー認証を通過しているかどうかを見分ける方法は？"
    answer: "認証済みの表示は通常 'Approvals' または 'Freigaben' です。'Performance Level'、'OEM Performances'、'Recommendations'、'OEM Compatibility'、'Suitable' と表示されているオイルは、実際には認証を通過していないことが多いため避けるべきです。"
  - question: "同じ認証のオイルを比較する際、重要なパラメータは何ですか？"
    answer: "比較すべきは Flash Point（高いほど良い）、100°C粘度（同じ認証・同じ番手なら低いほど良い）、40°C粘度（低いほど良い）、Noack（低いほど良い、推奨10以下）、HTHS（高いほど高回転に強い、2.9以上が望ましい）、TBN（オイル耐久性、8以上が望ましい）、Pour Point（-50度以下ならPAO含有率が高い可能性）。"
  - question: "エンジンオイル中の『カルシウム』成分はどのような影響がありますか？"
    answer: "カルシウムは洗浄剤ですが、LSPIを誘発しやすいです。そのため API SP/SQ 認証を通過したオイルではカルシウム含有量が通常 2000ppm 以下に制限され、一部をマグネシウムで代替して洗浄剤としています。"
  - question: "PAO と GTL 基油とは何ですか？"
    answer: "PAO（ポリαオレフィン）と GTL（Gas-to-Liquid）はどちらも最高級の合成基油です。PAO は通常少量のエステルを加えて添加剤と融合させます。GTL は天然ガスをフィッシャー・トロプシュ法で変換したもので、性能は PAO に近いです。"
  - question: "おすすめのエンジンオイルブランドは？"
    answer: "日常使用にはモービル (Mobil)、シェル (Shell)、カストロール (Castrol) などの大手ブランドが選べますが、偽物防止のため代理店経由で購入することを推奨します。性能重視ならリキモリ (Liqui Moly)、モチュール (Motul)、アムズオイル (AMSOIL)、レッドライン (Redline)、ワコー (Wako)、ラヴェノール (Ravenol) が選択肢です。台湾製では台塑、国宏、各メーカー純正オイルがあります。"
  - question: "購入時に避けるべきオイルブランドは？"
    answer: "国内の混合オイル、海外小規模工場に委託したブランド、見たことのないブランドは避けるべきです。知らないブランドに出会った場合は Mercedes-Benz Bevo サイトで検索し、認証数が10未満なら購入は推奨されません。"

## 主要な認証

### API 認証
- **SN Plus/SP**：LSPI テストの合格が要求される
- **SQ (ILSAC GF-7)**：2025年の新規格。LSPI保護、定時チェーン耐摩耗性、燃費経済性が強化され、さらに硫酸塩灰分 ≤0.9%に制限し、GPFを保護する

### メーカー認証

| 認証 | HTHS | TBN | Noack |
|------|------|------|------|
| **API SP/SQ** | >2.6 | なし | なし |
| **AECA A3/B4** | >3.5 | >10 | <13 |
| **AECA A5/B5** | 2.9-3.5 | >8 | <13 |
| **AECA C3** | >3.5 | >6 | <13 |
| **MB 229.5** | >3.5 | >10 | <10 |
| **MB 229.51/52** | >3.5 | >6 | <10 |
| **VW 502/505** | >3.7 | >7 | - |
| **VW 504/507** | >3.5 | >6 | <11 |
| **BMW LL-98** | >3.5 | >9.5 | - |
| **BMW LL-01** | >3.5 | >10 | - |
| **BMW LL-01 FW** | >3.0 | >9.5 | - |

## 認証の重ね合わせ（スタッキング）

- **229.5**：耐摩耗性基準は高いが、LSPI防止テストはない
- **API SP/SQ**：耐摩耗性基準は低いが、LSPI防止テストがある
- **229.5とAPI SP/SQの両方の認証を持つ場合** → 耐摩耗性に優れ、かつLSPI防止機能を持つ
- **認証の重ね合わせに関する参考情報**：[Lubrizol 八角形](https://online.lubrizol.com/relperftool/pc.html)

## 認証表示の文字ゲーム

**認証を取得していることを示す表記：**
- Approvals
- Freigaben

**認証を取得していないことを示す表記（使用を避けるべき）：**
- Performance Level
- OEM Performances
- Recommendations
- OEM Compatibility
- Suitable

## 【同一認証】オイル比較指標

- **フラッシュポイント (Flash Point)**：エンジンオイルの引火点。高いほど良い。
- **100°C粘度**：同じ認証、同じ粘度グレードの場合、低い方が望ましい。
- **40°C粘度**：低いほど良い。
- **Noack**：オイル揮発率。低い方が望ましく、10以下が推奨される。
- **HTHS**：高いほど高回転に耐え、低いほど燃費が良い。運転モードに応じて選択し、2.9以上が望ましい。
- **TBN**：オイルの耐久性。8以上が望ましい。
- **MRV**：低い方が良い（台湾では無視可能）。
- **CCS**：低い方が良い（台湾では無視可能）。
- **流動点 (Pour Point)**：最低流通温度。ベースオイルの種類を大まかに判断する指標で、-50度以下はPAO含有量が高い傾向がある。
- **VI**：粘度指数。参考価値なし。

## ベースオイルの種類

- **ポリエステル (Polyol Ester)**：最高級品。高回転に耐え、潤滑性が良いが、劣化が非常に速い。赤外線ピーク 1710>2
- **アルキルナフタレン (ANs)**：最高級品。稀少で、エステル類を代替しPAOと組み合わせて調製剤として使用される。赤外線ピーク 740-890
- **PAO**：最高級品。通常、添加剤に溶け込ませるために少量のエステル類が加えられる。赤外線ピーク 720>1
- **GTL**：最高級品。天然ガスをF-T化学合成したもので、性能はPAOに近く、赤外線ピーク 720<=1
- **XTL**：石炭や有機物を天然ガスに変換し、その後F-T合成で潤滑油とするもの。
- **VHVI**：水素化分解鉱物油。精製された鉱物油。
## 添加剤成分

- **カルシウム (Ca)**：洗浄剤。LSPIを誘発しやすい。SP/SQ認証済みは通常2000ppm未満である。
- **マグネシウム (Mg)**：洗浄剤。LSPIを誘発しやすいカルシウムの代替品。
- **モリブデン (Mo)**：耐摩耗剤
- **リン (P)**：耐摩耗剤
- **亜鉛 (Zn)**：耐摩耗剤。化合物名はZDDPと呼ばれ、潤滑性が良好である。
- **ホウ素 (B)**：耐摩耗剤
- **タングステン (W)**：耐摩耗剤
- **チタン (Ti)**：耐摩耗剤

## 推奨ブランド

### 日常使用向け **(世界各地に偽物が非常に多いため、正規代理店製品の購入を推奨します)**
- **美孚 Mobil**
- **キャスロール Shell**
- **キャストロール Castrol**

### 高回転域向け
- **リキリ・モリー Liqui Moly**
- **モーテル Motul**
- **アンソ AMSOIL**
- **レッドライン Redline**
- **ワコ Wako**
- **ジールマン Ravenol**

### 台湾製品
- **台塑**
- **国宏 (出光)**
- **各自動車メーカー純正オイル**

### **避けるべきもの**
- 現地でブレンドされたオイル、海外の小規模工場がOEM供給したブランド。
- 初めて見るブランドについては、[Bevo](https://bevo.mercedes-benz.com/bevolisten/bevo-sheets-sort1.html)にブランド名を英語で検索し、「Count」欄位の合計が10以上の場合のみ購入する。
