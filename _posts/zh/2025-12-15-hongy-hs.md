## 選購背景

前幾年，家中進口車陸續換上了 [GreenRun 鋰鐵電池](/2025/greenrun.html)，無論是使用感受還是壽命表現都相當令人滿意。近期有車友的 HS 1.5T 原廠電池開始衰退，內阻逐漸升高，經過研究後也決定更換鋰鐵電池。

HS 原廠電池為風帆 6-QW-63MG，屬於 **LN2** 規格。雖然 HS 的電池底座支援較大的 LN3 規格，但若改用 LN3 規格，原廠的隔熱套便無法沿用，因此最終決定維持 LN2 規格。

在台灣，LN2 規格與日規 B24、D23 面臨同樣的問題：市面鋰鐵電池選擇有限，**多數產品來自個體戶，或未標示工廠資訊**。雖然中國市場上的 12V 鋰鐵品牌眾多，但部分品牌常被批評容量虛標、做工粗糙。經過比較後，最終選擇深耕鋰鐵啟動電池多年，專為歐美品牌提供鋰電池 OEM/ODM 解決方案的製造商 **弘毅 (Houny)**。

[![弘毅 Houny 金階 LN2 85Ah 汽車啟動鋰鐵電池規格表](/images/LN2_HY_Type.webp)](/images/LN2_HY_Type.webp)

弘毅 2025 年改版之後，產品線分為金階、藍階、綠階三個系列，這次購入的是 85Ah 金階。同系列 LN2 規格還有 100Ah 可選，但考量到車友的 HS 屬於全原廠設定，僅作為假日用車，並無重度電系改裝需求，加上鋰鐵放電深度 (Depth of Discharge, DoD) 更深，可用容量是鉛酸的 1.6 倍，**85Ah 鋰鐵實際可用電量相當於 130Ah 鉛酸**，已相當足夠。

## 安裝作業

### 準備工具

| 工具名稱 | 用途與備註 |
| :--- | :--- |
| **10mm [L型套筒](/images/L-socket.webp)** | 拆卸電池底座固定螺絲與正負極端子必備 |
| **[OBD記憶保存器](/2025/memory-saving-battery.html)** | 不斷電更換用，防止行車電腦與音響設定流失 |
| **搭電電源** | 任意12V電池或救車行動電源，用來供電給 OBD 記憶保存器 |
| **[電池把手](/images/battery_handle.webp)** | 輔助提取（原廠電池無把手，強烈建議準備以節省體力） |

### 安裝步驟
> 以下為實際操作流程紀錄，安裝完畢後無任何故障碼。

[![MG HS 1.5T 汽車電池 DIY 不斷電安裝步驟圖解](/images/LN2-instruction.webp)](/images/LN2-instruction.webp)

| 步驟 | 動作說明 | 注意事項與細節 |
| :---: | :--- | :--- |
| **1** | **接上記憶保存器** | 將搭電電源接上 **OBD記憶保存器**，並插入車內的 OBD 接口。 |
| **2** | **車輛休眠** | 關閉所有車門，靜待 5 分鐘讓車輛電腦完全進入休眠狀態。 |
| **3** | **拆卸底座** | 使用 10mm 套筒解開電池底座固定螺絲，移除螺絲及壓板。 |
| **4** | **爭取操作空間** | 將舊電池 **往車頭方向推**，使其脫離卡榫，讓正極處的空間變大以便操作。 |
| **5** | **斷開端子** | 依序鬆開電池的 **負極**，接著再鬆開 **正極** 端子。 |
| **6** | **替換電池** | 套入[電池把手](/images/battery_handle.webp)取出舊電池，並放入全新的鋰鐵電池。 |
| **7** | **接回端子** | 依序鎖上 **正極**，接著再鎖上 **負極** 端子。 |
| **8** | **固定新電池** | 將新電池 **往駕駛座方向推**，確認緊密卡入底座的固定卡榫。 |
| **9** | **鎖回底座** | 將底座壓板與螺絲穩固鎖回。 |
| **10** | **完成安裝** | 移除 OBD 記憶保存器，發動車輛確認一切正常。 |

## 完工與心得

本次實際安裝耗時約 15 分鐘，MG HS 1.5T 引擎室空間充足，是相當好更換的車型。

[![MG HS 1.5T 引擎室安裝弘毅 LN2 鋰鐵電池實際視角](/images/LN2_HY_Install1.webp)](/images/LN2_HY_Install1.webp)

車友之所以決定更換為鋰鐵電池，主要看上鋰鐵的以下核心優勢：

| 優勢面向 | 詳細說明 | 預期效益 |
| :--- | :--- | :--- |
| **充放電性能** | [充電效率高](/images/mobile01-d8c9d0d9da6aecd839d3e3fbe4be1398.webp)，能有效減輕引擎負擔；[放電反應快](/2025/lithium-vs-lead-analysis.html)，電系運作更穩定。 | 延長車內發電機壽命，並改善怠速品質。 |
| **環境適應性** | 具備[高低溫穩定](/images/mobile01-8aeff577382fa233baa6f5da8eaead0b.webp)特性，磷酸鐵鋰結構安全。 | 耐高溫且無爆燃風險，提升引擎室用電安全。 |
| **物理特性** | 減重約 50%（原廠約 17kg，鋰鐵僅 8kg），自放電率極低。 | 輕量化提升操控感，**極低自放電率非常適合假日用車**。 |
| **油耗驗證** | [高充電效率減少能量浪費](/2025/car-bms.html)，發電機運轉充電的時間大幅縮短。 | 根據[研究](/images/mobile01-be90eec739eb64e5d45690809d60a24e.webp)顯示，更換後可實質優化車輛的油耗表現。 |
