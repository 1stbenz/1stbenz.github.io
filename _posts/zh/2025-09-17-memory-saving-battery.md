# 更換電池時保持供電（記憶保存）

> 依據 ISO 26262 及 UN R13/79 等國際規範，汽車電子系統在【斷電或重新上電】，必須回到「安全狀態」以避免危險。

在 Toyota 或 Honda 上，斷電通常只需重新設定時鐘。但若車輛已使用一段時間，便需留意**一些記憶設定遺失的後果**。

| 記憶參數 | 記憶清除後的現象 |
| :--- | :--- |
| <nobr><B>節氣門預設開度</B></nobr> | ECU 失去對應積碳狀態的補償記憶，退回原廠預設開度。這會導致實際進氣量不足，引發**怠速忽高忽低、嚴重抖動，甚至在停等或滑行時瞬間熄火**。 |
| <nobr><B>怠速控制</B></nobr> | 失去怠速空氣流量的學習值。在系統尚未完成重新學習前，會出現**怠速不穩、轉速異常浮動**的狀況。 |
| <nobr><B>變速箱換檔邏輯</B></nobr> | 清除過去依據駕駛習慣與油門開度記錄的參數。短期內會感覺變速箱**換檔時機改變、頓挫感明顯或銜接生硬**，需行駛一段時間讓電腦重新適應。 |
| <nobr><B>燃油修正值</B></nobr> | 噴油量長期修正值歸零。在 ECU 根據氧感知器數據重新建立最佳空燃比模型前，短期內可能會出現**油耗變差、引擎輸出動力不穩定**的現象。 |
| <nobr><B>方向盤角度感測器</B></nobr> | 失去方向盤轉向中心點的校正基準。可能會點亮儀表板故障燈，並**導致 ESP (車身動態穩定) 或 ABS (防鎖死煞車) 系統無法正常運作**。 |

> 這些設定在行駛一段時間後，會自動重新學習，但仍有機會造成車輛異常

既然斷電會有上述困擾，為何翻閱許多原廠維修手冊，SOP仍是「斷電更換」？
主因在於**原廠配有專用診斷電腦**。斷電後，原廠技師能即時清除故障碼、做感測器校正，甚至能在系統防盜鎖死時，重新執行防盜鑰匙的同步與授權。

相比之下，一般店家與DIY車主**沒有原廠權限的電腦**，**用「記憶保存器」不斷電更換，就是最保險的做法**。不僅能保留ECU的學習參數，還可避免IMMO異常或電子模組損毀，降低拖回原廠「開通」或「重刷」的可能性。

## 不斷電DIY注意事項：

## 1. 正極端子仍然帶電（最重要）

一般情況下，斷開電池後，正極夾子會變成無電的金屬。  
然而，若透過 OBDII 接口或點煙器插座連接外部電源（記憶保護器），正極夾子仍然保持 12V 電壓。

- **風險**：若鬆動的正極夾子接觸到車身、引擎或避震塔等金屬部位，會造成短路，可能燒斷記憶保護器的保險絲、OBDII 接口的保險絲，甚至損壞 ECU（電腦）。  
- **解決方法**：在移動電池前，立即用**厚乾布、橡膠手套或塑膠袋包裹正極夾子，並用膠帶或束帶固定**，避免接觸金屬。

---

## 2. 不要開門或操作車內電器

- **風險**：大多數記憶保護器提供的電流非常低。如果開門，頂燈或儀表螢幕會亮起，瞬間的電流需求可能耗盡記憶保護器或燒斷其保險絲，導致車輛斷電。  
- **解決方法**：事先將駕駛座車窗搖下（避免鎖車），關閉頭燈、收音機與冷氣，全程保持車門緊閉。

---

## 3. 確認 12V 插座（若使用點煙器方式）

- **風險**：許多現代車輛在熄火後會切斷點煙器/12V 插座的電源。如果插座在拔鑰匙後沒有電，記憶保護器就無法反向供電。  
- **解決方法**：先測試插座在未插鑰匙時是否有電（例如插入手機充電器）。若插座斷電，必須改用 OBDII 接口方式。

---

## 4. 連接穩定性

- **風險**：在搬運沉重電池時，記憶保護器的夾子或插頭容易被碰鬆。即使斷開一瞬間，記憶也會消失。  
- **解決方法**：確保記憶保護器插頭牢固，並將線材整理好，避免在操作時被勾到。

---

## 5. 安裝後：BMS 重置

即使不斷電維持了各項設定，你仍需處理電池管理系統（BMS）。

- **原因**：不斷電僅能保存記憶，卻無法告訴車輛已更換新電池。系統會認為還在使用舊電池。  
- **解決方法**：安裝新電池後，需透過診斷工具或特定按鍵程序來「註冊」或「編碼」新電池，確保發電機正確充電。
