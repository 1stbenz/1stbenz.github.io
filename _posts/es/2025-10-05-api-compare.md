  - question: "¿Cuál es la postura principal del artículo sobre las nuevas certificaciones de aceite?"

  - question: "¿Qué mejoras ofrece la certificación API SQ en el control de depósitos del pistón (IIIH)?"
    answer: "La certificación API SQ exige una puntuación mínima para los depósitos del pistón de 4.0 puntos, lo que representa un aumento respecto a la ≥3.5 puntos requerida por la API SP, demostrando así una capacidad de limpieza superior."
  - question: "¿Cuál es el requisito de la certificación API SQ para el aumento del índice de viscosidad (IIIH)?"
    answer: "La certificación API SQ requiere un aumento del índice de viscosidad ≤125%, lo que representa una exigencia más estricta que los ≤150% de la API SP, indicando que el aceite se desestabiliza con menos facilidad al aumentar su viscosidad a altas temperaturas."
  - question: "¿Qué mejoras presenta la certificación API SQ en las pruebas de volatilidad (IIIH)?"
    answer: "La certificación API SQ exige una volatilidad ≤60 mL, comparado con el límite ≤80 mL del API SP. La menor cantidad de evaporación ayuda a reducir el consumo de aceite."
  - question: "¿Qué nuevos estándares existen para la certificación API SQ en el control de depósitos a bajas temperaturas (VH)?"
    answer: "La certificación API SQ requiere una puntuación de deposición del pistón ≥6.5, lo que es más alto que los ≥6.0 requeridos por la API SP y ofrece un mejor efecto limpiador a baja temperatura."
  - question: "¿Qué mejoras ofrece la certificación API SQ en protección contra el desgaste (IVB)?"
    answer: "La certificación API SQ exige un desgaste ≤50 μm, lo que es más estricto que los ≤60 μm de la API SP y proporciona una mejor protección del motor contra el desgaste."
  - question: "¿Cuál es la diferencia entre la certificación API SQ y la API SP en cuanto a la protección contra LSPI?"
    answer: "La certificación API SQ no solo supera las pruebas de LSPI, sino que también incluye una prueba adicional de simulación de GDI de alta presión, proporcionando una protección más completa contra el encendido prematuro a bajas velocidades."
  - question: "¿Cuál es el requisito de la prueba de depósitos en turbina (X) para la certificación API SQ?"
    answer: "La certificación API SQ requiere una puntuación de depósitos en turbina ≥7.0 puntos, que es más alta que los ≥6.0 puntos requeridos por la API SP y permite un control más efectivo del encrustamiento en el compresor turbo."
  - question: "¿Cuál es el objetivo de mejora en la eficiencia del combustible (VIE) para la certificación API SQ?"
    answer: "El objetivo de la certificación API SQ es una mejora en la eficiencia del combustible ≥2,5%, correspondiente al estándar ILSAC GF-7, lo que representa un avance respecto a los ≥2,2% de la API SP."
  - question: "¿Qué cambios hay en los requisitos de la certificación API SQ para la bombeabilidad al arranque en frío (MRV)?"
    answer: "La certificación API SQ exige un viscosidad ≤60.000 cP a -40 °C para ciertos grados, una temperatura más baja que el -35 °C requerido por la API SP, mejorando así las prestaciones de arranque en temperaturas extremadamente bajas."
  - question: "¿Qué normas establece la certificación API SQ para el contenido de cenizas sulfatadas (SAPS)?"
    answer: "La certificación API SQ exige que el contenido de cenizas sulfatadas sea ≤0,90 % en peso y enfatiza la compatibilidad con GPF/DPF (filtros de partículas de gasolina/diesel), lo cual es favorable para proteger los sistemas de emisiones."
  - question: "¿Hasta qué porcentaje de compatibilidad con el combustible etanólico certificado por la norma API SQ se puede alcanzar?"
    answer: "La certificación API SQ admite hasta una compatibilidad con el combustible etanólico E85, que es superior a los E20 admitidos por la normativa API SP."
  - question: "¿Cuáles son los principales cambios en la especificación de API al evolucionar desde SN hasta SP?"
    answer: "De SN a SP, se agregaron principalmente pruebas para LSPI (pre-ignición por baja velocidad) y depósitos del turbo, con el objetivo de brindar soporte integral para motores GDI (inyección directa) turbocargados."
  - question: "¿Cuáles son las características principales de la evolución de los estándares API desde SP hasta SQ?"
    answer: "De SP a SQ, todos los umbrales de prueba se han reforzado y se ha añadido soporte para combustible E85, sistemas de emisiones GPF/DPF y vehículos híbridos, correspondiendo al estándar ILSAC GF-7."

### Evolución de las especificaciones


- **API SM → SN**: Se introducen pruebas VH e IVB para mejorar el control del desgaste y de los depósitos.  
- **SN → SP**: Se añaden las pruebas LSPI y de depósito en turbina, ofreciendo soporte completo para motores GDI con turbo.  
- **SP → SQ**: Se refuerzan todos los umbrales de prueba, habilitando E85, filtros de partículas (GPF/DPF) y vehículos híbridos, alineados con el estándar ILSAC GF‑7.
