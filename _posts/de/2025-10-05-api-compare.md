- **Neue Motoröl-Zertifizierung: Keine einzelne Norm ist rückläufig.**
- **Die Behauptung, dass „das Neue schlechter als das Alte" sei, ist eindeutig Pseudowissenschaft und dient nur der Lagerbestandsauflösung.**

| Testparameter (Sequence) | API SM (2004) | API SN (2010) | API SP (2020) | API SQ (2024) |
|----------------------|----------------|----------------|----------------|----------------|
| **Kupferabrieb (IIIH)** | Keine (IIIG) | Keine explizite Anforderung | ≥3.5 Punkte | ≥4.0 Punkte |
| **Viskositätsanstieg (IIIH)** | ≤275% | ≤200% | ≤150% | ≤125% |
| **Verdunstungsverluste (IIIH)** | Keine (IIIG) | Keine explizite Anforderung | ≤80 mL | ≤60 mL |
| **Kontrollierung niedriger Temperaturabrieb (VH)** | Kupferabrieb ≥4.0 Punkte (VG) | Kupferabrieb ≥5.0 Punkte | Kupferabrieb ≥6.0 Punkte | Kupferabrieb ≥6.5 Punkte |
| **Schmiermittelverträglichkeit (IVB)** | **Abrieb ≤120 μm** | **Abrieb ≤90 μm** | **Abrieb ≤60 μm** | **Abrieb ≤50 μm** |
| **LSPI-Schutz (IX)** | Keine | Keine | LSPI-Test bestanden | LSPI-Test + Hochdruck-GDI-Simulation bestanden |
| **Turboladerabriebstest (X)** | Keine | Keine | Turboladerabrieb ≥6.0 Punkte | Turboladerabrieb ≥7.0 Punkte |
| **Kraftstoffeffizienz (VIE)** | Verbesserung ≥1,1% (VIB) | Verbesserung ≥1,5% | Verbesserung ≥2,2% | Verbesserung ≥2,5% (ILSAC GF-7) |
| **Zündungspumpenverträglichkeit bei niedrigen Temperaturen (MRV)** | -25 °C, Viskosität ≤60.000 cP | -30 °C, Viskosität ≤60.000 cP | -35 °C, Viskosität ≤60.000 cP | -40 °C, Viskosität ≤60.000 cP (einige Klassen) |
| **Hohe Temperatur und hohe Scherfestigkeit (HTHS)** | ≥2,9 cP @150 °C | Gleiche wie oben | Gleiche wie oben | Gleiche wie oben, aber Schwankungen ≤5% |
| **Sulfatashalt (SAPS)** | ≤1,2 Gew.% | ≤1,2 Gew.% | ≤0,90 Gew.% | ≤0,90 Gew., Betonung der Kompatibilität mit GPF/DPF |
| **Verträglichkeit mit Ethanolkraftstoff** | E10 | E10 | E20 | E85 (höchster Wert) |




### Spezifikationenentwicklung


- **API SM → SN**: Einführung von VH und IVB-Tests zur Verbesserung der Verschleiß- und Ablagerungskontrolle.  
- **SN → SP**: Hinzufügen von LSPI- und Turbelablagerungstests, umfassende Unterstützung für GDI-Turboanlagen.  
- **SP → SQ**: Stärkung aller Testgrenzwerte mit Unterstützung für E85, GPF/DPF-Hybridfahrzeuge im Einklang mit ILSAC GF-7.
